// features

Everything you need

Built for industrial engineers who need reliability, not bloat.

⚡

Modbus RTU & TCP

Full support for RTU over serial RS-232/RS-485 and TCP/IP. Reads all four register types: coils, discrete inputs, holding and input registers.

📊

Real-time monitoring

Live register values with configurable poll rates from 100ms. Visual charts, data logging and export to CSV.

🔍

Serial Monitor

See every raw byte on the line in real time. Inspect TX/RX traffic, spot protocol errors and debug communication issues without extra tools.

🔎

Slave & Address Scan

Automatically scan for active slave IDs on a network or scan all register addresses in a device — essential when working with unknown hardware.

🎨

Conditional colors & scaling

Highlight register values by threshold with custom colors. Apply scaling and value labels so raw numbers show meaningful engineering units.

🔄

Auto-reconnect

Automatically reconnects after a dropped connection — no manual intervention needed. Keeps long-running monitoring sessions alive without babysitting.

💾

Workspace save / load

Save your entire session — all windows, poll configs, profiles and settings — and reload it instantly next time.

🌐

Multilingual UI

Full interface available in multiple languages. Switch languages without reinstalling — built for international engineering teams.

🔒

100% offline

No cloud, no telemetry, no internet connection required. Your industrial data stays on your machine, always.
